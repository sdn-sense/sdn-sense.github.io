## Overview

**Nokia SR-OS** (Service Router Operating System) is the network operating system for Nokia (formerly Alcatel-Lucent) 7750 SR and related service router platforms. Support for Nokia SR-OS in SENSE SiteRM is currently **in development** and is expected to be available in 2026.

| Property | Value |
|---|---|
| **Ansible `network_os`** | `sense.nokiasros.nokiasros` *(planned)* |
| **Ansible Collection** | [sense-nokiasros-collection](https://github.com/sdn-sense/sense-nokiasros-collection) *(in development)* |
| **VLAN Creation** | No *(planned)* |
| **BGP Control** | No *(planned)* |
| **BGP Multipath** | No *(planned)* |
| **QoS (network-level)** | No *(planned)* |
| **Ping / Traceroute** | No *(planned)* |
| **Viz in MRML** | No *(planned)* |

---

## Current Status

The `sense-nokiasros-collection` repository exists as a placeholder in preparation for future development. The collection does not yet contain implementation code. No SENSE capabilities are available for Nokia SR-OS at this time.

If your site requires Nokia SR-OS integration, contact the SENSE team at [sense-user@googlegroups.com](mailto:sense-user@googlegroups.com) to discuss requirements and timelines.
